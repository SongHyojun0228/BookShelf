해결 방법은 두 가지가 있습니다:

1. 상세 정보가 필요한 순간(모달을 열 때) 알라딘 API를 한 번 더 호출해서 쪽수를 알아온다. (조금 코드가 길어지지만 쪽수가 무조건 필요하다면 선택!)
2. 쪽수 표시는 삭제하고, 당분간 다른 정보(별점, 출판사 등)로만 만족한다.
