# 4. 모니터링 삼총사(Prometheus, Grafana, cAdvisor) 구축

서버가 잘 돌아가고 있는지, 갑자기 CPU가 치솟지는 않는지 **눈으로 직접 확인(시각화)**하는 것은 데브옵스의 핵심입니다. 이를 위해 우리는 업계 표준인 '모니터링 삼총사'를 사용합니다.

---

## 🧐 1. 왜 모니터링을 하나요? (Observability)
"건강 검진"과 똑같습니다. 
- 서버가 언제 죽을지 미리 예측하고,
- 사용자가 몰릴 때 컴퓨터를 더 빌려야 할지 판단하며,
- 장애가 났을 때 원인이 무엇인지(메모리 부족? CPU 과부하?) 빠르게 파악하기 위함입니다.

---

## 🧙‍♂️ 2. 모니터링 삼총사의 역할

### ① cAdvisor (Container Advisor) - "CCTV / 센서"
- **역할**: 도커 컨테이너의 실시간 상태를 수집합니다.
- **설명**: 각 컨테이너가 CPU를 얼마나 쓰는지, 메모리를 얼마나 먹는지 1초 단위로 측정해서 내보냅니다.
- **이미지**: `gcr.io/cadvisor/cadvisor`

### ② Prometheus (프로메테우스) - "DB / 수집가"
- **역할**: cAdvisor가 내보낸 숫자 데이터를 일정 시간마다 읽어와서 저장합니다.
- **설명**: 일종의 '시계열 데이터베이스'입니다. 과거부터 지금까지의 상태를 모두 기록해 둡니다.
- **이미지**: `prom/prometheus`

### ③ Grafana (그라파나) - "대시보드 / 시각화"
- **역할**: 프로메테우스에 저장된 숫자 데이터를 아주 예쁜 그래프로 그려줍니다.
- **설명**: 우리가 실제로 보는 **화려한 모니터 화면**이 바로 그라파나입니다.
- **이미지**: `grafana/grafana` (초기 ID/PW: admin / admin)

---

## 🛠️ 3. 설정 파일 (Configuration)

### [docker-compose.yml]
삼총사를 동시에 띄우기 위해 기존 설정에 덧붙입니다.
```yaml
    # 1. 정보원 (cAdvisor)
    cadvisor:
      image: gcr.io/cadvisor/cadvisor:latest
      container_name: cadvisor
      ports:
        - "8081:8080"
      volumes:
        - /:/rootfs:ro
        - /var/run:/var/run:ro
        - /sys:/sys:ro
        - /var/lib/docker/:/var/lib/docker:ro

    # 2. 데이터 창고 (Prometheus)
    prometheus:
      image: prom/prometheus:latest
      container_name: prometheus
      ports:
        - "9090:9090"
      volumes:
        - ./prometheus.yml:/etc/prometheus/prometheus.yml:ro

    # 3. 예술가 (Grafana)
    grafana:
      image: grafana/grafana:latest
      container_name: grafana
      ports:
        - "3000:3000"
```

### [prometheus.yml]
프로메테우스가 어디서 데이터를 긁어와야 할지 알려주는 파일입니다.
```yaml
global:
  scrape_interval: 5s # 5초마다 데이터를 수집해!

scrape_configs:
  - job_name: "docker-containers"
    static_configs:
      - targets: ["cadvisor:8080"] # cAdvisor(8080포트)에서 데이터를 가져와!
```

---

## 🚀 4. 운영 프로세스
1. `docker-compose up -d` 로 삼총사를 실행합니다.
2. **[localhost:3000]** 에 접속합니다.
3. **Data Source** 를 `Prometheus`로 설정하고 URL에 `http://prometheus:9090`을 넣습니다.
4. 나만의 대시보드를 만들거나 고수들의 템플릿(ID: 193 등)을 가져와서(Import) 적용합니다.

**이제 당신은 마우스 클릭만으로 서버의 건강 상태를 체크하는 '간지' 나는 엔지니어가 되었습니다!** 😎🦾
